# License Plate Recognition System (OpenCV & Tesseract)

A robust Python-based application that detects and recognizes vehicle license plates using image processing techniques and Optical Character Recognition (OCR). This project supports static images, real-time webcam streams, and database integration.

## 🚀 Features
- **Static Image Processing:** Detects plates from high-resolution images.
- **Real-time Detection:** Captures and processes license plates directly from a webcam feed.
- **OCR Integration:** Uses Pytesseract to convert image data into string format.
- **Database Support:** Automatically logs detected plates with timestamps into an SQL Server database.

## 🛠 Tech Stack
- **Language:** Python
- **Computer Vision:** OpenCV
- **Image Processing:** Numpy, Imutils
- **OCR Engine:** Tesseract OCR
- **Database:** SQL Server (pypyodbc)
